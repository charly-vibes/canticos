canticos

Cánticos, invocaciones, escrituras — scripts de uso personal.

  • claude-dashboard

    python html analytics

    Dashboard HTML local de uso de Claude Code. Escanea sesiones, calcula costos estimados, genera gráficos de actividad, uso de herramientas, cache hit rate, y más. Todo offline, sin servidores.

  • to-projector

    bash ffmpeg vaapi

    Convierte video a MP4 compatible con proyectores portátiles (H.264 8-bit + AAC estéreo). Auto-detecta si necesita re-encoding o puede copiar streams. Soporta VAAPI para encoding por hardware, selección de audio por idioma y quemado de subtítulos.

  • projector-queue

    bash batch

    Procesa una cola de videos en ~/Downloads/MOVIES/ a través de to-projector. Idempotente: los archivos convertidos se mueven a DONE/, los ya procesados se saltan.

  • transcribe

    bash whisper ffmpeg

    Transcribe audio a texto usando whisper-cli. Convierte a WAV 16kHz mono, corre el modelo y deja un .txt junto al archivo original.

  • ci-usage

    python gh ci

    Reporte de uso de GitHub Actions para un repositorio. Muestra qué workflows y runs consumen más minutos, desglose por trigger y branch, y opcionalmente minutos facturables por OS con multiplicadores.

Instalación

git clone https://github.com/charly-vibes/canticos.git
cd canticos
./install

Crea symlinks en ~/.local/bin. Si ya existe un archivo, lo respalda como .bak.
